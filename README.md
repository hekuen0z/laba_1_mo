### Результы
В данной лабораторной работе была использована Логистическая регрессия, обученная с помощью оптимизатора SAGA, для классификации изображений шрифтов notMNIST после их нормализации и преобразования в вектор признаков. 
Максимально достигнутая точность с использование линейной регресии на больших данных - ~83% c количесвом итераций в 200 и силой регуляции равной 0.01.
Увеличение количества итераций (max=500) и уменьшение значения силы регуляции (max=10 в -4 степени) привели к существенному увеличению времени выполнения без значительного увеличения точности модели.
